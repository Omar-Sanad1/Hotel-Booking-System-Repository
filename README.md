# Hotel-Booking-System-Repository 

Backend API for managing hotel rooms, reservations, payments, and customer authentication using JWT.

## Technologies
- ASP .Net Core Web API
- Entity Framework Core (Code First)
- SQL Server
- JWT Authentication
- AutoMapper
- BCrypt For Password Hashing
- Swagger

## Features
- User Registeration & Login
- JWT Authentication
- Room Booking
- Reservation Management
- Hotel Management (CRUD)
- Room Management with Availability Status
- Room Search & Filtering (Hotel, Type, Price, Capacity, Availability)
- Automatic Price Calculation based on nights
- Payment Management
- Room Reviews & Ratings
- Customer Account Management

## Architecture
- Onion Architecture :
                      - Domain Layer (Core)
                      - Repository Layer
                      - Service Layer
                      - API Layer
  -DTOs

  ## How To Run
  1. Clone Repository
  2. Update Connection String
  3. Run Migration
  4. Run Project
